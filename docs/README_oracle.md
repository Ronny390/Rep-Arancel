# Guía de Migración y Conectividad con Oracle Database (cx_Oracle / oracledb)

Esta sección contiene toda la documentación necesaria para levantar la estructura del Arancel de Venezuela en una instancia de **Oracle Database** y realizar cargas, consultas y reportes utilizando Python.

---

## 1. Estructura de Base de Datos en Oracle
Oracle utiliza tipos de datos optimizados para almacenamiento empresarial. Las conversiones aplicadas en este repositorio son:
- **`SERIAL PRIMARY KEY`** (PostgreSQL) se convierte a **`NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY`** en Oracle.
- Los textos largos e interpretados (descripciones anidadas y notas legales extensas) se declaran como **`CLOB`** (Character Large Object) en lugar de `TEXT` para evitar la limitación de 4000 caracteres del tradicional `VARCHAR2`.
- Las llaves foráneas se configuran con **`ON DELETE CASCADE`** para mantener la consistencia al purgar niveles de la jerarquía arancelaria.

El script de creación de tablas DDL está en:
`sql/oracle/arancel_venezuela_oracle.sql`

---

## 2. Migración Automatizada con Python (`oracledb`)
Hemos sustituido el antiguo conector `cx_Oracle` por el moderno **`oracledb`** (versión oficial de Oracle), el cual funciona en modo **Thin** (100% puro Python) y no requiere instalar binarios de **Oracle Instant Client** en el sistema.

### Instalación de dependencias:
```bash
pip install oracledb
```

### Ejecución del script de migración:
1. Asegúrate de configurar las variables de entorno de conexión o edita los parámetros por defecto en el script:
   ```bash
   set DB_USER=system
   set DB_PASSWORD=tu_clave
   set DB_DSN=localhost:1521/FREE
   ```
2. Ejecuta el script:
   ```bash
   python scripts/migrar_db_oracle.py
   ```
   *El script se encargará de crear las tablas en Oracle, leer los archivos CSV limpios de la carpeta `datos_csv/` e insertarlos con control transaccional perfecto.*

## 3. Ejemplo de Consulta en Python (cx_Oracle / oracledb)
Para consultar la descripción y notas de un producto usando Oracle desde Python:

```python
import oracledb

connection = oracledb.connect(user="system", password="password", dsn="localhost:1521/FREE")
cursor = connection.cursor()

# Buscar caballo en Oracle
query = """
SELECT sp.codigo, sp.descripcion, sp.aec 
FROM subpartidas sp 
WHERE sp.descripcion LIKE :buscar
"""
cursor.execute(query, buscar="%CABALLO%")
for row in cursor:
    print(f"Código: {row[0]} | Arancel: {row[2]}%\nDescripción: {row[1]}\n")

connection.close()
```
